# Snake Game (Java)

This is my first project in Java. A simple reimagined of Snake Game.

# FEATURES
* Classic Snake Game
* Smooth Controls
* Input locker to avoid rapid key input
* Snake Game Physics


## How is works?
1. Ensure that you have a Java Development Kit (JDK) installed.
2. Close this repository.
3. Run the Snake.java in your IDE (IntelliJ).
